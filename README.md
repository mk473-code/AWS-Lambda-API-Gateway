# AWS Lambda + API Gateway

## Project Overview

This project demonstrates a serverless REST API using AWS Lambda and API Gateway.

The Lambda function processes requests and returns a JSON response through an API endpoint.

## Technologies Used

- Python
- AWS Lambda
- Amazon API Gateway

## Features

- Serverless architecture
- JSON response handling
- API endpoint integration
- Cloud-based execution

## AWS Services Used

- AWS Lambda
- Amazon API Gateway

## Project Workflow

1. Created Lambda function.
2. Developed Python handler.
3. Configured API Gateway trigger.
4. Connected API Gateway to Lambda.
5. Tested API endpoint.
6. Verified JSON response.

  "message": "Hello world, welcome to AWS Lambda!"
}
